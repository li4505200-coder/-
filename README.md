# shmetro-loadtest-gis-v8

This repository hosts the built static site for the Shanghai metro power-supply GIS demo.

Deployment target:
- GitHub Pages from `main` branch `/docs` folder, or
- Vercel / Netlify by importing this repository.

Build notes:
- Vite base path has been changed to `./` so assets load correctly from subpaths.
- The publishable site is under `docs/`.
